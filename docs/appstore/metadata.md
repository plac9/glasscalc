## Description

### Short Description (170 chars)

### Full Description (4000 chars max)

PrismCalc is a beautifully designed calculator that combines powerful everyday tools with a stunning glassmorphic interface. Built exclusively for iOS 18, it showcases the latest Apple design language with animated mesh gradients, smooth transitions, and delightful micro-interactions.

**CORE CALCULATOR**
• Clean, intuitive interface for basic arithmetic
• Large, easy-to-tap glass buttons with haptic feedback
• Percentage and sign toggle functions

**HISTORY (Pro)**
• Save and review past calculations
• Lock important entries for quick reference

**TIP CALCULATOR (Pro)**
• Calculate tips from 0-100% with an elegant arc slider
• Quick presets: 15%, 18%, 20%, 25%
• Split the bill among multiple people
• See tip amount and total per person instantly

**BILL SPLIT (Pro)**
• Divide any bill equally among friends
• Include or exclude tip in the split
• Perfect for group dinners and shared expenses

**DISCOUNT CALCULATOR (Pro)**
• Calculate sale prices instantly
• See both savings and final price
• Great for shopping and budget planning

**UNIT CONVERTER (Pro)**
• Length: miles, kilometers, feet, meters, and more
• Weight: pounds, kilograms, ounces, grams
• Temperature: Fahrenheit, Celsius, Kelvin
• Instant conversion as you type

**STUNNING DESIGN**
• 6 beautiful themes with animated mesh gradient backgrounds
• True glassmorphism with layered materials
• SF Symbol animations for delightful interactions
• Floating tab bar with iPad sidebar support
• Smooth zoom transitions between views

**WIDGETS & SIRI**
• Home Screen widgets in 3 sizes
• Control Center quick actions (iOS 18)
• Siri Shortcuts for hands-free calculations
• "Hey Siri, calculate 18% tip on $50"

**ACCESSIBILITY**
• Full VoiceOver support with descriptive labels
• Dynamic Type for adjustable text sizes
• Reduce Motion support for sensitive users
